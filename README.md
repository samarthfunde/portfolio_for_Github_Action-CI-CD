# Portfolio Website - GitHub Actions CI/CD Practice

This repository is created for learning and practicing CI/CD using GitHub Actions.  
It contains basic workflows to understand automation concepts such as build, test, and deployment.

## Purpose
- Learn GitHub Actions fundamentals
- Understand CI/CD pipelines
- Automate build and test workflows
- Practice DevOps concepts using GitHub Actions

## What This Repository Covers
- GitHub Actions workflows
- YAML syntax for pipelines
- Triggering workflows on push and pull requests
- Running jobs on GitHub-hosted runners
- Basic automation for applications

## Technologies Used
- GitHub Actions
- YAML
- Git
- Linux (GitHub runners)

## How It Works
1. Code is pushed to the repository
2. GitHub Actions workflow is triggered
3. Jobs run automatically on GitHub-hosted runners
4. Workflow results are visible in the Actions tab

## Learning Objective
The main goal of this repository is to gain hands-on experience with CI/CD pipelines and understand how automation improves software delivery.

## Status
This is a learning and practice repository. Workflows will be updated as learning progresses.
